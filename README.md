# CVNets
经典和最新的计算机视觉网络的`Pytorch`实现，并在简单数据集上进行测试

**数据集：**

* [ ] MINST(手写数字识别)
* [ ] Cifar-10
* [ ] Cifar-100

数据集分两种调用形式：

1. 直接调用`torchvision.datasets`中的接口(无需手动下载数据)；
2. 自己写`dataloader`调用(需要手动下载数据，但方便理解数据读取过程)。

**网络模型**

* [ ] AlexNet
* [ ] VGGNet系列
* [ ] ResNet系列
* [ ] VIT(Vision Transformer)
* [ ] MLP
